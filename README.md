### 이메일(Email) 디자인을 위한 인라인 CSS 자동 처리 프로세스

외부 `<link>` 요소 또는 `<style></style>`에 적용된 스타일 코드를
각각의 HTML 요소에 인라인 스타일로 적용한 후, `<link>` 요소 또는 `<style></style>`는 제거합니다.

```sh
$ gulp email # 단일 프로세스 처리
$ gulp       # 지속적 관찰 업무 추가
```
